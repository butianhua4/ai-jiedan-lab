# SEO Growth Roadmap

## Stage 1: Technical Foundation

Entry conditions:
- Project builds locally.
- Public routes are stable.

Exit conditions:
- `npm run lint` passes.
- `npm run seo:check` passes.
- `npm run build` passes.
- Sitemap, robots, and canonicals are healthy.
- Internal link health is 100.

Autonomous tasks:
- crawlability checks
- canonical checks
- robots checks
- indexability reports

Manual confirmation:
- domain changes
- DNS changes
- production hosting changes

Success metrics:
- zero build blockers
- zero orphan pages
- no draft leakage

## Stage 2: Indexing Warmup

Entry conditions:
- Technical Foundation is complete.
- q pages, cluster pages, and blog pages exist.

Exit conditions:
- priority q pages submitted to GSC/Bing
- cluster pages submitted
- no orphan pages

Autonomous tasks:
- generate GSC submission lists
- generate Bing/IndexNow push plans
- keep sitemap priority reports current

Manual confirmation:
- Search Console ownership changes
- manual URL inspection quotas

Success metrics:
- growing discovered URL count
- fewer "discovered not indexed" pages over time

## Stage 3: Traffic Activation

Entry conditions:
- Search engines have discovered priority pages.
- impressions begin appearing.

Exit conditions:
- titles and descriptions for high-potential pages are tuned
- FAQ schema is added where useful
- CTR improvement opportunities are tracked

Autonomous tasks:
- q title optimization plan
- q description optimization plan
- FAQ schema tasks
- top 50 q optimization list

Manual confirmation:
- claims about actual rankings or revenue

Success metrics:
- impressions rising
- first clicks confirmed
- query data imported without fake values

## Stage 4: Conversion Layer

Entry conditions:
- SEO structure is stable.
- traffic or discovery signals exist, or monetization entry points are missing.

Exit conditions:
- service page exists
- hire/contact entry exists
- template and tool CTAs exist
- lead form plan exists

Autonomous tasks:
- add service page
- add CTA placeholders
- add template conversion improvements
- add conversion reports

Manual confirmation:
- pricing guarantees
- payment account setup
- ad account setup

Success metrics:
- clear service funnel
- measurable CTA clicks once analytics data is available

## Stage 5: Tool Expansion

Entry conditions:
- traffic and conversion paths are visible.
- tool opportunities are prioritized.

Exit conditions:
- useful calculators/checkers exist
- proposal and error tools are stronger
- tools link into q/blog/cluster pages

Autonomous tasks:
- tool planning reports
- low-risk UI additions
- internal links to tools

Manual confirmation:
- paid API costs
- user account systems
- data storage for sensitive inputs

Success metrics:
- more tool landing traffic
- repeat usage signals

## Stage 6: International Expansion

Entry conditions:
- Chinese structure is stable.
- US search market remains a target.

Exit conditions:
- `/en` entry is complete
- English q page framework exists
- English tools and affiliate slots are planned

Autonomous tasks:
- English q draft framework
- Chinese-to-English expansion plan
- international keyword plan

Manual confirmation:
- bulk English publication
- affiliate partnerships

Success metrics:
- English pages indexed
- US impressions begin appearing

## Stage 7: Scale to 10k

Entry conditions:
- quality gates are reliable.
- indexing and engagement data prove traction.

Exit conditions:
- query database exists
- staged publishing exists
- quality scoring blocks bad pages

Autonomous tasks:
- next 1000 q plan without publishing
- quality scoring reports
- staged publishing design

Manual confirmation:
- mass publication
- infrastructure cost increases
- paid data sources

Success metrics:
- scalable page generation without duplicate or low-quality content
- crawl budget is not wasted
